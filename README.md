## 🚧 Installation

1. Clone this repository :

```bash
git clone https://github.com/SupakunZ/Tesla_Clone.git
```

2. Navigate to the project folder and install dependencies :

```
cd Tesla_Clone
npm install
```

3. Launch the application in development mode :

```
npm run dev
```
